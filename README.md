# Haven Wellness Supplement Finder

An interactive web app to help users discover the right wellness protocols for their health goals.

## Features

- **Protocol Finder Quiz** - Guided questionnaire to recommend products based on health goals
- **Category Filtering** - Filter by Injury & Recovery, Gut Health, Longevity, Energy & Focus, Immunity, Metabolism, Heart Health
- **Product Comparison** - Select up to 3 products for side-by-side comparison
- **Search** - Find products by symptoms, names, or benefits
- **Expandable Product Cards** - View detailed information including benefits, target audience, and clinical evidence

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is ready to deploy on:
- **Vercel** - Connect your GitHub repo and deploy automatically
- **Netlify** - Drag and drop the `dist` folder after running `npm run build`
